<h2 id="english">🇬🇧 English</h2>

# Lira Studio - Premium Web Development Agency

A brutally fast, ultra-converting brutalist portfolio designed for elite web agencies. Built to score 100/100 on Google Web Vitals, it leverages modern React paradigms, smooth scrolling, and cinematic page transitions.

> **Live Demo:** [https://liraconversao.com.br](https://liraconversao.com.br) *(Example link)*

### 🚀 Core Features
- **Awwwards-Winning Aesthetics:** Brutalist typography, subtle neon accents, and magnetic UI interactions.
- **Cinematic Transitions:** Framer Motion-powered page curtain effects and scroll-linked parallax animations.
- **Extreme Performance:** Lazy loading, asynchronous asset decoding, and aggressive font preloading for instant First Contentful Paint (FCP).
- **Accessibility (A11y):** Full keyboard navigation support and rigorous ARIA labeling.
- **Infinite Marquee Ticker:** Dynamic text reveal and seamless looping banner components.

### 🛠 Tech Stack
- **Framework:** React 18 (Vite)
- **Routing:** React Router v6
- **Animations:** Framer Motion
- **Scroll Hijacking:** Lenis (Studio Freight)
- **Icons:** Lucide React
- **Deployment:** Vercel

### ⚙️ How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/mateusoliveiradev1/lira.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

<h2 id="português-brasileiro">🇧🇷 Português Brasileiro</h2>

# Lira Studio - Agência de Desenvolvimento Web Premium

Um portfólio brutalista incrivelmente rápido e focado em ultra-conversão, desenhado para agências web de elite. Construído para gabaritar o Google Web Vitals (100/100), utilizando paradigmas modernos do React, scroll fluido e transições cinematográficas.

> **Demo Online:** [https://liraconversao.com.br](https://liraconversao.com.br) *(Link de exemplo)*

### 🚀 Funcionalidades Principais
- **Estética Nível "Awwwards":** Tipografia brutalista, toques em verde-neon e botões magnéticos.
- **Transições Cinematográficas:** Efeito cortina (sweep) em mudanças de rota e animações de scroll parallax impulsionadas pelo Framer Motion.
- **Performance Extrema:** Lazy loading puro, decodificação assíncrona de mídia e pré-carregamento de fontes (Preload) para o máximo FCP.
- **Acessibilidade (A11y):** Suporte total a navegação por teclado e `aria-labels` estrategicamente inseridos para robôs de SEO.
- **Marquee Infinito:** Tickers contínuos de palavras-chave para criar dinamismo visual 24/7.

### 🛠 Tech Stack
- **Framework:** React 18 (Vite)
- **Roteamento:** React Router v6
- **Animações:** Framer Motion
- **Scroll Fluido:** Lenis (Studio Freight)
- **Ícones:** Lucide React
- **Hospedagem:** Vercel

### ⚙️ Como Rodar Localmente
1. Clone o repositório:
   ```bash
   git clone https://github.com/mateusoliveiradev1/lira.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
